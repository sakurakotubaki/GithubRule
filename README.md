# Githubのコミットのルールについてのドキュメント

|接頭語|意味|
|----|----|
|[F]|	Fix。バグ修正|
|[A]|	Add。新規追加|
|[U]|	Update。機能修正|
|[R]|	Remove。削除|
|[WIP]|	Work In Progress。作業中|

せっとう‐ご【接頭語】
語構成要素の一。 単独では用いられず、常に他の語の上について、その語とともに一語を形成するもの。 語調を整えたり、意味を添加したりする。 「お話」「こ犬」「御親切」などの「お」「こ」「御」の類。
